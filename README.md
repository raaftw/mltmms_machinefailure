## Project Overview

This project explores whether equipment failures can be predicted in advance using machine learning. The dataset contains daily sensor measurements, machine age, and several machine attributes (such as manufacturer, region, and maintenance vendor), along with recorded failure events. By analysing trends and patterns in these features, the goal is to build models that can provide early warning of failures and to identify which measurements are most strongly linked to impending failure.

## Research Question

For the analysis of this dataset we want to answer the following research question: **Can equipment failures be predicted in advance, and which measurement is the most indicative of impending failure?**

To make this question more concrete, we consider the following sub-questions:

- **To what extent can sensor data and machine age be used to predict upcoming failures?**
- **Do machine attributes such as manufacturer, region, or maintenance vendor influence failure likelihood?**
- **Are certain machine types more predictable than others based on the available features?**
